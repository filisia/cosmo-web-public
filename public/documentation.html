  • macOS 10.15 (Catalina) or newer
  • Built-in Bluetooth or compatible Bluetooth adapter
  • Bluetooth permissions for the application

üì¶ Installation for End Users

  1. Download CosmoBridge-Mac-Setup.dmg installer
  2. Open the .dmg file
  3. Drag "Cosmo Bridge" to the Applications folder
  4. Launch from Applications folder
  5. Grant Bluetooth permissions in System Preferences when prompted

üíª Web Development Guide

Quick Start Example

Basic Device Manager

class CosmoDeviceManager {
  constructor() {
    this.devices = new Map();
    this.ws = null;
    this.listeners = new Set();
  }
  
  connect() {
    this.ws = new WebSocket('ws://localhost:8080');
    
    this.ws.onopen = () => {
      console.log('Connected to Cosmo Bridge');
      this.requestDevices();
    };
    
    this.ws.onmessage = (event) => {
      const message = JSON.parse(event.data);
      this.handleMessage(message);
    };
  }
  
  setColor(deviceId, r, g, b) {
    this.ws.send(JSON.stringify({
      type: 'setColor',
      deviceId,
      color: { r, g, b }
    }));
  }
}

Educational Game Pattern

Simple Educational Game

class CosmoEducationalGame {
  constructor(deviceManager) {
    this.deviceManager = deviceManager;
    this.score = 0;
    this.currentTarget = null;
    
    // Listen for button events
    this.deviceManager.addListener(this.handleGameEvent.bind(this));
  }
  
  startGame(deviceIds) {
    // Set colors for each device
    const colors = [
      { r: 255, g: 0, b: 0 },   // Red
      { r: 0, g: 255, b: 0 },   // Green  
      { r: 0, g: 0, b: 255 }    // Blue
    ];
    
    deviceIds.forEach((deviceId, index) => {
      const color = colors[index % colors.length];
      this.deviceManager.setColor(deviceId, color.r, color.g, color.b);
    });
    
    this.nextTarget();
  }
  
  handleGameEvent(message) {
    if (message.type === 'buttonPress' && message.deviceId === this.currentTarget) {
      this.score++;
      console.log('Correct! Score:', this.score);
      this.nextTarget();
    }
  }
}

üìã Code Samples

Multi-Device Light Show

class CosmoLightShow {
  async rainbowPattern(devices, duration = 5000) {
    const colors = [
      [255, 0, 0], [255, 127, 0], [255, 255, 0],
      [0, 255, 0], [0, 0, 255], [75, 0, 130]
    ];
    
    const stepDuration = duration / colors.length;
    
    for (let i = 0; i < colors.length; i++) {
      const [r, g, b] = colors[i];
      devices.forEach(deviceId => {
        this.deviceManager.setColor(deviceId, r, g, b);
        this.deviceManager.setLuminosity(deviceId, 128);
      });
      
      await new Promise(resolve => setTimeout(resolve, stepDuration));
    }
  }
}

üîß Troubleshooting

Connection Problems

Issue: Cannot connect to WebSocket

Solution:

  • Ensure bridge application is running
  • Check that port 8080 is not blocked by firewall
  • Verify WebSocket URL is ws://localhost:8080
