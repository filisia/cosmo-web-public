üîß Manual Installation

macOS Installation

# Download the latest release curl -L -o cosmo-bridge.dmg https://github.com/your-repo/cosmo-bridge/releases/latest/download/cosmo-bridge-mac.dmg # Mount and install hdiutil attach cosmo-bridge.dmg cp -R /Volumes/CosmoBridge/CosmoBridge.app /Applications/ hdiutil detach /Volumes/CosmoBridge

Windows Installation

# Download the installer # Run the .exe file and follow the installation wizard # Or use winget (if available) winget install CosmoBridge

Linux Installation

# Download the AppImage wget https://github.com/your-repo/cosmo-bridge/releases/latest/download/cosmo-bridge-linux.AppImage # Make executable chmod +x cosmo-bridge-linux.AppImage # Run ./cosmo-bridge-linux.AppImage

üîç Troubleshooting

Connection Issues

If the connection test fails:
  • Make sure the Cosmo Bridge application is running
  • Check that port 8080 is not blocked by your firewall
  • Try restarting the bridge application
  • Check the bridge application logs for errors

Firewall Settings

# macOS - Allow incoming connections sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /Applications/CosmoBridge.app # Windows - Add firewall rule netsh advfirewall firewall add rule name="CosmoBridge" dir=in action=allow program="C:\Program Files\CosmoBridge\CosmoBridge.exe" # Linux - Allow port 8080 sudo ufw allow 8080

Port Already in Use

# Check what's using port 8080 lsof -i :8080 # Kill the process if needed kill -9 [PID]

üì± Connecting Devices

Once your bridge is running and connected:

  1. Turn on your Cosmo device
  2. Put it in pairing mode (usually by holding a button)
  3. The bridge will automatically detect and connect to your device
  4. Your device will appear in the web app
